AbstractModel
service in module core

Description

This abstract model should be inherited by all models using the TAMU-UI-Core. It exposes unwrapping capabilites. All abstracted methods can go here: (e.g. AbstractModel.myMethod = funciton() {} ) A model can then extend this my including "self = this;" and "angular.extend(self, AbstractModel);" in its contructor.

Usage

AbstractModel(data);

Parameters

ParamTypeDetails
dataobject

The models data, from the webservice, to be extended onto the model itself.

Returns

service

AbsrtactModel returns the AbstractModel

Example

The following must be added to a model for it to extending this AbstractModel 

 angular.extend(self, AbstractModel);
