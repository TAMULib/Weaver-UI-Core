tab
directive in module core

Description

The tab element directive uses the 'TabService' which in turn uses the ' settingsView' and the html file location to render the html view of the clicked tab and is set as activeTab. The tab value is stored as '$scope.tab' variable.

Dependencies

  • TabService

Usage

as element:
<tab>
</tab>

Directive info

  • This directive creates new scope.

Properties

  • $scope.tab

    This variable stores the string value of tab on the scope of link function.

Example

<tabs target="settingsView">
		<tab path="myview" html="views/myview.html">My View Tab</tab>
	</tabs>
