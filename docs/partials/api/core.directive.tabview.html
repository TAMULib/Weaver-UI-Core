tabview
directive in module core

Description

The tabview span element directive uses the 'TabService' which in turn uses the 'attr' object to get the html view and store it on the '$scope.path' variable

Dependencies

  • TabService

Usage

as element:
<tabview>
</tabview>

Directive info

  • This directive creates new scope.

Properties

  • $scope.path

    The $scope.path variable stores the html view of the active tab using the 'TabService'.

Example

<tabs target="settingsView">
		<tab path="myview" html="views/myview.html">My View Tab</tab>
	</tabs>
	<tabview class="settings-view" for="settingsView">
	</tabview>
