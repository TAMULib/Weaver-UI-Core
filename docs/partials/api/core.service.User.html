User
service in module core

Description

This User service must be injected into a controller class that require dependency on User model. This model provides the user credentials to a controller. Extends 'AbstractModel'

Dependencies

  • AbstractModel
  • WsApi

Usage

User(futureData);

Parameters

ParamTypeDetails
futureDataobject

The models data, from the webservice, to be extended onto the model itself.

Returns

service

User returns the User service

Example

myApp.controller('MyController', function (User) { 
	$scope.user = User.get();
}
