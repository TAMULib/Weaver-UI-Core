tooltip
directive in module core

Description

The tooltip element directive uses $timeout and $compile services. It has showTip hideTip This directive uses $compile services that compiles 'tipTemplate' variable into a template and produces a template function, which can then be linked to '$scope' and the template. This is then appended to the html body

Dependencies

  • $timeout
  • $compile

Usage

as attribute
<ANY tooltip>
   ...
</ANY>

Methods

  • $scope.hideTip()

    This method will use the $timeout service to cancel the tooltip message display,will set the boolean value of $scope.tipVisible to false and executes a function to set the boolean value of $scope.hidden boolean variable to true.

    Returns
    void

    returns void

  • $scope.positionTip($event)

    This method stores X and Y cordinates of the mouse move event and is stored on to $scope.tipStyles["top"] and $scope.tipStyles["left"] variables respectively to aid display of the tooltip message in that position.

    Parameters
    ParamTypeDetails
    $eventobject

    an object

    Returns
    void

    returns void

  • $scope.showTip()

    This method will set the $scope.hidden boolean value to false and will use the $timeout service to display the tooltip message for the duration of 500msec.

    Returns
    void

    returns void

  • $scope.toggleVisible()

    This method will use $timeout to cancel the tooltip message display and toggle the boolean values for *scope.tipVisible and $scope.hidden variables.

    Returns
    void

    returns void

Properties

  • $scope.hidden

    The $scope.hidden variable stores the boolean value 'true' if the tooltip message is displayed and 'false' otherwise.

  • $scope.showTimer

    The $scope.showTimer object uses the $timeout service to execute a function to toggle the boolean values of $scope.tipVisible and $scope.hidden variables depending on the duration provided in the function.

  • $scope.tip

    The $scope.tip variable stores the text message of the tooltip from the 'attr' objects tooltip property.

  • $scope.tipStyles

    The $scope.tipStyles object stores the x and y cordinate values for the 'mousemove' event inorder to display the tooltip text message.

  • $scope.tipVisible

    The $scope.tipVisible variable stores the boolean value depending on whether the $scope.tip variable is displayed or not.

  • tipTemplate

    This variable provides a template for the tooltip message.

Example

<tooltip title="The tooltip message">
</tooltip>
