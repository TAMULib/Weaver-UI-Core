AccordionService
service in module core

Description

The AccordionService can add remove panes from the 'openPanes' list using the 'paneID' passed as an argument. The 'closeAll' method closes all the panes in the 'openPanes' list.

Methods

  • AccordionService.add(id, close)

    This method adds an Accordion's close method to the openPanes store, using the id as a key.

    Parameters
    ParamTypeDetails
    idinteger

    The id of the open accrodion pane.

    closefunction

    The close method of the open pane.

    Returns
    void

    returns void

  • AccordionService.closeAll(id)

    This loops over the openPanes store and calls the close method on each member. If an id is passed into it then that pane will remain open.

    Parameters
    ParamTypeDetails
    id
    (optional)
    		integer

    An open Accordion pane's id.

    Returns
    void

    returns void

  • AccordionService.remove(close)

    This method removes an Accordion's close method to the openPanes store, using the id as a key.

    Parameters
    ParamTypeDetails
    closefunction

    The close method of the open pane.

    Returns
    void

    returns void

Properties

  • openPanes

    A private object to store open panes. The key should be the id of the open pane, and the value its close method.
