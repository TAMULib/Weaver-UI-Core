pane
directive in module core

Description

This pane directive uses the 'AccordionService' to open and close the pane based on

Dependencies

  • $location
  • $timeout
  • $anchorScroll
  • AccordionService

Usage

as element:
<pane>
</pane>

Methods

  • $scope.close()

    This method assigns 'false' to the boolean variable '$scope.expanded'.

    Returns
    void

    returns void

  • $scope.close()

    This method uses the $timeout services to execute the function which is to set the boolean value of '$scope.loading' to false after the given duration of 500ms.

    Returns
    Promise

    returns promise

  • $scope.open()

    This method stores the view path from 'attr' object on to the $scope.html variable. The AccordionService closes the previous opened pane based on the 'paneID' and the $location services replaces the existing URL with the new URL based on the query string '$scope.query'.

    Returns
    void

    returns void

  • $scope.toggleExpanded()

    A method uses the AccordionService to toggle the state of the pane clicked.

    Returns
    void

    returns void

Properties

  • $scope.query

    A $scope variable to store the 'query string' tag of pane directive.

Example

The following must be added to your application to use the accordion directive followed by the panes within it 

	<pane query="query_string_for_myview" html="views/myview.html">
		PANE TITLE
	</pane>
