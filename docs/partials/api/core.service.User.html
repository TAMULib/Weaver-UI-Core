User
service in module core

Description

This User service must be injected into a controller class that require dependency on User model. This model provides the user credentials to a controller. A model can then extend this by including "self = this;" and "angular.extend(self, AbstractModel);" in its contructor.

Dependencies

  • AbstractModel
  • WsApi

Usage

User(futureData);

Parameters

ParamTypeDetails
futureDataobject

The models data, from the webservice, to be extended onto the model itself.
