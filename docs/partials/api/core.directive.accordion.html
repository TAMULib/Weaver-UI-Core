accordion
directive in module core

Description

This accordion directive sets the tag "single-expand"=true which ensures that only one pane( one view) can be clicked open at a time.

Dependencies

  • AccordionService

Usage

as element:
<accordion>
</accordion>

Methods

  • $scope.closeAll(id)

    A method uses the AccordionService to close all open panes.

    Parameters
    ParamTypeDetails
    id
    (optional)
    		integer

    An open Accordion pane's id which you which should be left open.

    Returns
    void

    returns void

Properties

  • $scope.singleExpand

    A $scope set from the accordion's element atrribute of the same name. This indicated that the accordion will only allow one pane open at a time.

Example

The following must be added to your application to use the accordion directive followed by the panes within it 

	<accordion single-expand="true">
     </accordion>
